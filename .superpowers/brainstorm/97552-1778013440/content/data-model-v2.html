Data Model — Revised

Field names mapped directly from pokemontcg.io + TCGdex real API responses

Data strategy: pokemontcg.io is the primary card catalog source — it already embeds Cardmarket + TCGPlayer prices in each card response. TCGdex adds Italian/multilingual card names (query /it/cards/…) and detailed holo variant pricing. Daily cron fetches both and merges into MongoDB.
